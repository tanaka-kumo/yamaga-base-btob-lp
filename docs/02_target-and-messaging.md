# 02 ターゲット定義とメッセージング設計

## メインターゲット（中心読者）

**熊本県内の経営者・経営企画担当者**

- 中小〜中堅企業のオーナー経営者、二代目／後継者、経営企画・経営管理の責任者。
- 県内なので物理的な距離は近い（植木ICから車10分）。「遠くて行けない」障壁は低い。
- すでに YAMAGA BASE の名前は聞いたことがある層が多い。だが「自社がどう使うか」は未解像。
- 学び意欲・成長意欲は高いが、時間と人材が常に不足。投資対効果に敏感。

### この人が抱える「経営の悩み」（訴求の起点）

- 経営チームの目線がそろわない／中期戦略を腰を据えて考える時間がない。
- 幹部・後継者を育てたいが、体系的に経営を学べる場が県内に乏しい。
- 日常の延長では発想が広がらない。非日常で頭を切り替えたい。
- 採用・定着が厳しい。会社の魅力・成長環境を打ち出したい。
- 「学び」や「合宿」を外部に出すと費用も移動も重い。近場で完結させたい。

## サブターゲット（脇役として配慮）

- 県外/全国の人事・研修担当（ワーケーション型研修・合宿先を探している）。
- スタートアップ・新規事業担当（登記可能なシェアオフィス・コミュニティ）。

※ 初期LPは中心読者に最適化。サブ向けの導線は用意するが主役を食わない。

---

## メッセージング原則

1. **機能ではなく「経営課題の解決」を売る。** 「体育館があります」ではなく「日常を離れて経営チームが本音で語れる」。
2. **二重価値構造を明示する。** ＝「ワクワクする非日常（情緒）」×「成果と稟議の根拠（理性）」。
3. **固有の強みを一言で。** 「廃校まるごと貸せる」「ハーバードMBAの学びが県内で受けられる」は他にない武器。
4. **行動の心理的ハードルを下げる。** まずは視察・相談・資料、という軽い一歩を用意する。

## 課題 → 利用シーン → 提供価値 翻訳表（LP本文の素材）

### A. 経営合宿（主役①）

| 読者の課題 | YAMAGA BASE での利用シーン | 提供価値（言い換え） |
|-----------|--------------------------|-------------------|
| 中期戦略を腰を据えて議論する時間がない | 1〜2泊で経営チームが合宿。会議室＋宿泊＋食が同一敷地で完結 | 移動ゼロで「考え抜く」に集中できる |
| 役員の目線がそろわない | 廃校の非日常空間で日常の役職・空気をリセット | 本音で対話でき、合意形成が進む |

| 都会の貸会議室は味気ない | 灯籠照明・古墳モチーフ等、山鹿の文化が宿る空間 | 記憶に残る「ここでやった」体験 |
| 合宿の段取りが面倒 | ワンストップ（会場/宿泊/食/アクティビティ）で手配 | 幹事の負担を最小化 |

## 稟議突破のために必ず載せる要素（理性側）

- 利用シーン別の**進め方の流れ**（問い合わせ→視察→プラン設計→当日→振り返り 等）。
- **費用感の目安**（確定値は `facts-and-figures.md`。未確定なら「要見積/目安」で表現）。
- **アクセスの良さ**（植木ICから10分、空港から40分、新幹線新玉名から25分）。
- **施設スペック**（収容人数・設備・宿泊定員 等：要確認項目が多い）。
- 可能なら**利用企業の声・事例**（※許諾と事実確認が前提。なければ載せない）。

## トーン

- 「秘密基地に集う、挑戦する経営者」。前向き・骨太・少し遊び心。
- 県内経営者に対しては「地元から、世界水準の学びと挑戦を」という誇りに訴える。
- 既存ブランド：bY（be you / beyond you / beside you）、iReaction、「秘密基地」を世界観の核に。
