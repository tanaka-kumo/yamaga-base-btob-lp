## 3. YAMAGA BASE とは（30秒で要点）

**見出し（H2）案：**
1. 廃校を、まるごと。経営者のための基地へ。
2. 会議も、宿泊も、運動も、食も。ぜんぶ同じ敷地で。
3. 山鹿の文化が宿る、唯一無二の「秘密基地」。

**リード文案：**
- YAMAGA BASE は、2024年に開業した廃校活用の複合施設です。
  旧・千田小学校の校舎と校庭がまるごと、働き・学び・集うための場所に生まれ変わりました。

**本文（要点／機能羅列でなく「なぜここなのか」）：**
- **廃校をまるごと使える。** 校舎・体育館・運動場・校庭まで、ひとつの敷地に。日常から完全に切り離された空間が、思考と対話の質を変えます。

- **「出会いが化学反応を起こす」場所。** iReaction を掲げ、地方から新しい挑戦を生む——その思想が施設全体を貫いています。

**ミニ統計（確定数値のみ）：**

## 4. 活用法①：経営合宿（主役）

**見出し（H2）案：**
1. 経営合宿｜日常を離れ、チームで「考え抜く」一泊二日。
2. 役員合宿を、記憶に残る「ここでやった」体験に。
3. 移動ゼロで、戦略に集中する。経営合宿という選択。

**リード文案：**
- 戦略策定も、チームビルディングも。会議室・宿泊・食・運動が同じ敷地にそろうから、
  「移動」と「段取り」に奪われていた時間を、まるごと「考える」に使えます。

**本文（docs/02 A表を本文化｜課題→価値）：**
- **腰を据えて議論する時間がない →** 1〜2泊の合宿で、会議・宿泊・食事をすべて同一敷地で。移動ゼロで「考え抜く」に集中できます。
- **役員の目線がそろわない →** 廃校の非日常空間が、日常の役職や空気を一度リセット。本音で語り合え、合意形成が前に進みます。
- **チームの一体感が薄い →** 体育館・運動場・BBQ・キャンプで「仕事以外の時間」を共有。余白の時間が関係性を変えます。
- **都会の貸会議室は味気ない →** 灯籠の灯り、山鹿の文化が宿る空間。「あの場所でやった」という記憶ごと、合宿が成果になります。
- **合宿の段取りが面倒 →** 会場・宿泊・食・アクティビティをまとめて手配。幹事の負担を最小限に。

**使える設備（カード化想定）：**
- 貸会議室（CLASS ROOM）／宿泊（ドミトリー・戸建・キャンプ）／体育館・運動場・ジム／BBQ・キャンプエリア／旧給食室のカフェ
