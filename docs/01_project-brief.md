# 01 プロジェクトブリーフ

## 背景

YAMAGA BASE は、2024年4月に熊本県山鹿市鹿央町の旧・千田小学校跡地をリノベーションして開業した複合施設。コワーキング、シェアオフィス、宿泊（ドミトリー/戸建/キャンプ）、運動施設（体育館・運動場・ジム）、貸会議室、配信・キッチンスタジオなどを備える。「廃校×秘密基地」という唯一無二の空間で、地方からのイノベーション創出（iReaction）を掲げている。

法人向けの収益軸は現在おおむね4つ：

1. **bYコミュニティ（MBA×AIの会員制）** — 主にtoC寄りだが、一朶の雲のように法人登記して活用する例もある。
2. **法人向け経営合宿での施設利用** ← 本LPの主役①
3. **中原功寛代表が講師を務める経営セミナー（MBAエッセンスを学べる）** ← 本LPの主役②
4. **施設を会場としたイベント開催**

これらが既存サイトでは複数ページに分散し、「法人がここで何をできるか」が一望できる入口がない。

## 課題（このLPで解く）

1. 認知の質 — 名前は知られていても具体的な利用イメージ・訴求できる課題が伝わっていない。
2. 稟議突破力 — 意思決定者が社内提案に使える「効果・根拠・費用感・進め方」が不足。
3. 体験価値の解像度 — ワクワクが伝わりきっていない。

## ゴール / KPI（仮）

- **主要コンバージョン**：法人問い合わせ・視察予約・資料請求。
- **サブ**：経営セミナーの関心登録、合宿プラン相談。
- 数値目標は運営と要すり合わせ（`facts-and-figures.md`【要確認】）。

## スコープ

- **In**：法人向け単一LP（静的）、主役2軸の訴求、CTA設計、計測フック。
- **Out（初期）**：予約システム連携、多言語化、CMS化、既存サイトの改修。
  （いずれも中長期で検討余地ありとして `decision-log.md` に記録）

## 成果物形態

独立した静的LP（単体HTML + CSS + 最小JS）。別ホスティング想定。
既存サイトのサブパス（例 `/business`）に将来統合する可能性は残すが、初期は独立稼働を優先。

## ターゲット（詳細は 02）

中心読者：**熊本県内の経営者・経営企画担当者**。
県内のビジネスパーソンには一定の認知があるため、「知っている」から「使ってみたい・社内に提案したい」へ引き上げる役割を担う。

## 主役マネタイズ軸の位置づけ

| 軸 | 本LPでの扱い | 想定する読者の動機 |
|----|------------|----------------|
| 経営合宿での利用 | 主役① | 非日常空間で経営チームの合宿・戦略策定・チームビルディングをしたい |
| 経営セミナー（MBA / 中原代表） | 主役② | 経営知見を体系的に学びたい・幹部育成をしたい |
| bYコミュニティ | 脇役（導線として紹介） | 継続的な学び・法人活用 |
| イベント会場利用 | 脇役（導線として紹介） | セミナー/発表会/研修の会場として |
