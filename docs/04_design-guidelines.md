# 04 デザインガイドライン

## デザインの目的

「ワクワクする秘密基地」の情緒と、「経営者が信頼して稟議に乗せられる」理性を**同居**させる。子どもっぽくなりすぎず、堅すぎず。骨太で前向き、少しの遊び心。

## キーワード

- 廃校 × 秘密基地 × 経営の学び舎
- 山鹿の文化（灯籠の灯り、来民団扇、鹿の角＝再生の象徴）
- iReaction（出会いが化学反応を起こす）
- 余白・自然・本気で考える静けさ

## ブランド資産（既存サイトとの連続性）

- ロゴ／マーク：bY マーク（既存 `logo_by.svg`）。
- タグライン：be You. beyond You. we will stand bY. at base de Yamaga.
- ヘキサゴン（六角形）モチーフが既存KVで使われている → アクセントに踏襲可。
- ※ 正式なブランドガイド・カラーコード・ロゴデータは運営から入手する（`facts-and-figures.md`【要確認】）。

## カラー（暫定方針 / 要正式確定）

確定するまでは以下の方向性で仮置きし、CSS変数で一元管理する。
- ベース：余白を活かした白〜オフホワイト。
- メイン：信頼感のある深めのトーン（ネイビー／チャコール系）。
- アクセント：灯籠の灯りを思わせる温かみ（アンバー／テラコッタ系）。
- 木・自然を感じるニュートラル（ウッド/グリーンを差し色）。

```css
:root {
  --color-base: #faf8f4;      /* オフホワイト（仮） */
  --color-ink: #1f2630;       /* 文字・主要色（仮ネイビーチャコール） */
  --color-accent: #e0852f;    /* 灯籠の灯り（仮アンバー） */
  --color-sub: #3f6b5a;       /* 自然のグリーン（仮） */
  --color-muted: #6b7280;     /* 補助テキスト */
}
```

## タイポグラフィ

- 日本語：Noto Sans JP（本文）／見出しは太めウェイトで力強さを。
- 英字・数字：可読性の高いサンセリフ（Inter 等）。
- 見出しは大きめ・余白広め。経営者向けに「詰め込みすぎない」品。
- 本文は16px以上、行間ゆったり（1.7〜1.9）。

## レイアウト原則

- **モバイルファースト**（県内経営者のスマホ閲覧を想定）。
- 1カラム基調。セクションごとに余白でリズムを作る。
- 写真は大きく使い、廃校×非日常の空気を伝える（alt必須）。
- 数値・流れ・料金は「図解／カード／タイムライン」で視認性を上げる（テキストの壁にしない）。
- CTAボタンは一目で分かる配色・十分なタップ領域（モバイルで48px以上）。

## 写真・ビジュアル方針

- 実際の施設写真を主役に（廃校外観、灯籠照明の内装、体育館、キャンプ、スタジオ）。
- 画像は最適化（WebP/適切なサイズ）してパフォーマンスを確保。
- 正式な写真素材は運営から入手（`reference/` のURLは確認用であり成果物に直リンクしない方針）。

## アクセシビリティ

- 文字色と背景のコントラスト比 4.5:1 以上。
- 見出し階層（h1→h2→h3）を正しく。
- 画像 alt、リンクテキストは具体的に。
- フォーカス可視・キーボード操作可。

## パフォーマンス

- 静的・軽量。重いライブラリは避ける。
- フォントは表示最適化（font-display: swap）。
- 画像遅延読み込み（loading="lazy"）。
