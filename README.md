# YAMAGA BASE｜法人向けLP プロジェクト

熊本県山鹿市の廃校活用施設 **YAMAGA BASE** の、法人（BtoB）集客に特化したランディングページを制作・運用するためのリポジトリです。制作主体は一朶の雲株式会社。

## このリポジトリの目的

YAMAGA BASE を「経営合宿」「経営セミナー（MBA）」を中心とした法人利用先として、**熊本県内の経営者・経営企画担当者**に訴求するLPを作ること。単なる施設紹介ではなく「自社のこの課題をここで解ける」と担当者がイメージでき、かつ社内稟議に乗せられるページを目指します。

## まず読むもの

1. **`CLAUDE.md`** — プロジェクトの中核コンテキスト。AIで作業する場合は最初にこれを読ませる。
2. **`docs/01_project-brief.md`** — 全体方針。
3. **`docs/facts-and-figures.md`** — 確定事実と「要確認」事項の台帳。**数字や社名はここを正とする。**

## ディレクトリ

| パス | 役割 |
|------|------|
| `CLAUDE.md` | AI/人間共通の中核コンテキスト |
| `docs/` | 戦略・設計・事実台帳（一次資料） |
| `prompts/` | AIに渡す初期プロンプト集 |
| `src/` | LP本体（成果物） |
| `reference/` | 既存サイトから取得した一次情報の控え（**成果物に流用しない**） |

## AIで作業を始める（Claude Code 等）

```
1) CLAUDE.md を読み込む
2) prompts/00_kickoff-prompt.md の内容を最初の指示として渡す
3) タスクに応じて copywriting / build-lp プロンプトを使う
```

## 運用ルール（要点）

- 方針を変えたら `docs/decision-log.md` に必ず追記する。
- 未確認の数字・社名・料金は創作せず、本文ではプレースホルダにし `facts-and-figures.md` の【要確認】に積む。
- 既存サイトの文章をコピペしない（自分の言葉で書く）。
